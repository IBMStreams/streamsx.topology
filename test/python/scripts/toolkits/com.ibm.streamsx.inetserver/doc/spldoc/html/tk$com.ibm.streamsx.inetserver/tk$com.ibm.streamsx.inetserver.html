General Information

The Internet Toolkit provides support for common internet rest protocol and web socket server protocol. This toolkit is a split from streamsx.inet 2.9.6

This toolkit separates its functionality into a number of namespaces:
  • com.ibm.streamsx.inet.rest: Operators that embed a HTTP server to provide a REST-style API to streaming data.
  • com.ibm.streamsx.inet.wsserver: Operators that embed a WebSocket server to expose streaming data as WebSocket messages.
